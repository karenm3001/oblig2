GIT

Git er et verktøy for versjonskontroll og kodedeling. Folk har mulighet å jobbe sammen, alle kan bruke samme filene. Hvert arbeidstre i Git er et fullt repository med all historikk og full sporing av revisjoner. Man er ikke avhengig av nettverk eller en sentral tjeneste. Man kan jobbe offline og endre på filene hvor mange ganger man vil.

GitHub

GitHub er en Git-tjeneste som lar deg enkelt behandle og bla i gjennom kode og informasjon som har blitt lastet opp av deg selv og andre. I tillegg til dette har den funksjoner som Github Pages som enkelt lar deg sette opp sider hvor du kan legge opp tekst og bilder som kanskje er relevant til prosjektet du har lagt inn på Github.
